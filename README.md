# DocuMind — Production RAG System

A production-grade Retrieval-Augmented Generation (RAG) system built progressively over 4 weeks.

## Project Structure

| Week | Focus | Status |
|------|-------|--------|
| [Week 1](./week-1/) | PDF Ingestion, Chunking, Embeddings, ChromaDB | Done |
| [Week 2](./week-2/) | Hybrid Retrieval, BM25, Reranking, Multi-LLM Generation | Done |
| Week 3 | Evaluation Pipeline (RAGAS) | Upcoming |
| Week 4 | FastAPI + Streamlit Frontend | Upcoming |

## Tech Stack
Python, PyMuPDF, ChromaDB, sentence-transformers, OpenAI, Anthropic, Mistral, Cohere Rerank, BM25
